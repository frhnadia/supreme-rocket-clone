# Supreme Rocket Landing Page Clone 
This project is a front-end website inspired by the Supreme Rocket Maintenance landing page design from: https://rocketman.learnwagtail.com/ . 
This project focuses on layout structuring and responsive design using HTML5 and CSS3.
